This is a selection sort.
Selection sort is the sort that selects the smallest data from the data array and sends it forward.

Characteristics of Selection Alignment Algorithm
Advantages: The number of data transfers is determined in advance.
Disadvantages: 1. Not satisfied with stability.
         2.That is, when there is a record with the same value, the relative position may be changed.

Time Complexity of Selection Sort
Best : O(n^2) , Avg : O(n^2) , Worst : O(n^2)
